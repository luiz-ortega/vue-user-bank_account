# VueJS User/Bank-Accounts Dashboard

A simple dashboard for add, edit, remove users and register bank accounts made with [VueJS](https://vuejs.org/v2/guide/). 

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
